ユーザーに様々な種類の「ひんやり」をお届けする2Dアドベンチャーゲーム

君と氷の旅路

ここは冷たい氷雪の領域
(ひんやりどころじゃないとか言わない)

足を踏み入れた人間は、寒さのあまり体が氷像へと変化するという

…

主人公が外を歩ける時間は限られている

制限時間内に、塔で待つ家族のもとへ辿り着こう

…

Enterで、氷雪の世界へ足を踏み入れる

### **推しポイント**  
**様々なひんやり要素**

>世界観/ビジュアル
>* 氷雪に覆われた地域が舞台。寒くてひんやり。
>* 切ないストーリー。心がひんやり。
>  
>システム/ギミック
>* 全ステージを通してプレイヤーが徐々に凍っていく時間制限制。緊迫感でひやひや。
>* とある条件下で不思議な世界が見える。怖くてひんやり。
>* 冷たい氷をアイテムで溶かして進む。ひんやり。

**全4ステージそれぞれで異なるギミックを実装し、遊びごたえのあるゲームに**

>第一ステージ（大量の放棄されたトロッコがある雪原）
>* 基本的な操作方法やアイテムの使い方を覚えてもらうチュートリアル的ステージ
>
>第二ステージ（一面凍り付いた海）
>* 物に当たるまで停止できない氷の床ギミック
>
>第三ステージ（塔の一室）
>* ある条件に合わせたアイテムの置き換えギミック
>
>第四ステージ（塔の最上階）
>* 狭い視界の謎解きギミック

**クオリティを突き詰めたビジュアル**
>* マップ全体は冷たくさびしい印象のまま、ステージはそれぞれ異なる雰囲気に
>* プレイヤーの歩行モーション
>* タイトルとエンディングのアニメーション

作り込まれた世界観を短時間でプレイできるゲームに詰め込みました！

### **世界観設定**
世界はなんかすごい災害によって気候変動が起こり、一部の地域が氷に覆われた(氷雪エリア)。

生き残った人類は限られた面積の「居住区」と呼ばれる区域に集まり暮らしている。

氷雪エリアは氷雪におおわれ、踏み入った人間は徐々に身体が凍結し、最終的には氷像と化してしまう。

この氷雪エリアは年々拡大を続けている。

居住区の環境はヨーロッパちっくな雰囲気。

人類が住めるような環境とはいえ、居住区の気温はとても低い。

人々の服装はヨーロッパみのある厚着で、外に出る際はコート、マフラー、帽子、手袋と言った防寒具は欠かせない。

人々の生活に必要なものは全て居住区で揃う。居住区の人々は滅多なことが無い限り氷雪エリアに足を踏み入れることは無い。

### **ストーリー解説** 
主人公は居住区に住んでいる。昔は双子の妹と両親とペットと一緒に暮らしていたが、住んでいた場所が氷雪エリアに飲み込まれた。両親は仕事に行っていたため、双子とペットだけ居住区に避難することができた。

妹はある日両親を探すと言って居住区を出て行ってしまう。主人公はしばらくペットと暮らしていたが、妹から手紙が届く。内容は「両親を見つけた、また家族みんなで暮らそう。氷雪エリアの『塔』の最上階にきて」。それを読んだ主人公は妹を探しに行く決心をする。

主人公はペットと一緒に氷雪エリアに踏み込む。

**↓ここからが本編の出来事↓**

主人公たちは塔を目指す。

主人公たちはギミックを解きながら塔の最上階を目指す。

最上階を目指す過程でペットが死にそうになったり、主人公の身体が徐々に凍っていったりする。

最上階で主人公は幸せそうな家族の氷像と出会う。すぐ隣にはペットを救うための充電器がある。

どちらも凍り付いてしまっている。だが、氷を溶かすアイテムの使用回数はあと一回。

　１．もし氷を溶かすアイテムを充電器に使用すれば、ペットは復活する。しかし主人公は家族と氷像になる
 
　２．もし氷を溶かすアイテムが氷像に使えば、主人公は家族と再会する

>(裏設定)
>
>氷雪エリア化は神様的な存在が意図的に引き起こしたものである。
>
>主人公たちの両親は研究者である。両親はこの拡大する氷雪エリアから逃げられないと思い、宇宙への逃避を考えていた。そのために氷雪エリアのすぐ近くにある、放棄されたロケット発射場にて研究を行っていた。人類が宇宙に逃げることを赦さない神様的存在は、主人公の両親もろと>も、ロケット発射場を氷雪エリアに飲み込ませた。
>
>主人公の妹は、主人公以上に両親のいない現実に絶望していた。主人公と二人で暮らすより、家族みんなで半永久的に傍に居続けることを望み、主人公に手紙を送った。
